Part 2 – Battle game

This part is a simple battle game using the HP API.

The user can:
- summon 2 random characters
- characters can be students or teachers

Each character has:
- students → 500 HP
- teachers → 800 HP

During the battle:
- a random spell is selected from the API
- random damage is applied
- teachers deal slightly more damage

The fight is turn-based and continues until one character reaches 0 HP.

There is also a small chance to cast **Avada Kedavra**, which instantly defeats the opponent.
